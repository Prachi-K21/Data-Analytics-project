# Data Analytics Project

This project was completed as part of my Data Analytics learning journey and portfolio development.

## Overview
This project demonstrates a complete data analytics workflow including:

- Web Scraping
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights

## Project Tasks

### Task 1: Web Scraping
Collected quote data from Quotes to Scrape using Python and BeautifulSoup.

### Task 2: Data Cleaning
Performed:
- Missing value analysis
- Duplicate removal
- Data type verification

### Task 3: Exploratory Data Analysis
Analyzed:
- Most quoted authors
- Tag distribution
- Quote frequency

### Task 4: Data Visualization
Created visualizations using:
- Matplotlib
- Seaborn

## Dataset Information

| Feature | Description |
|----------|------------|
| Quote | Quote text |
| Author | Quote author |
| Tags | Associated tags |

Dataset Size:
- Rows: 100
- Columns: 3

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- BeautifulSoup
- Matplotlib
- Seaborn

- Albert Einstein appears most frequently.
- Dataset contains 3 missing values in Tags.
